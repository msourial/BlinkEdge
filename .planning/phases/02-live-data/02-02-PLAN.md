| Boundary | Description |
|----------|-------------|
| SW runtime → Cache storage | Service worker reads/writes precache and runtime caches. No sensitive data in cache (precached assets are static). |
| SW update flow → Browser | New SW detected → prompts user to refresh. No auto-activation (user must tap). |
| App → ~offline page | Navigation fallback serves cached page when offline. |
