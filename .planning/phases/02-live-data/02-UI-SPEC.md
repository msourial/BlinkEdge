**New Phase 02 screens (all within phone frame):**

### Screen 1: Match Selection (pre-camera)
- Full-height vertical list of matches with gradient backdrop fallback (no camera active)
- Scrollable `overflow-y-auto` with `contain: layout paint`
- Each item: 4-line card (home team / score / away team / time + status)
- Tapping a match:
  1. Sets the selected matchId in context
  2. Starts camera flow (calls existing `startARCamera`)
  3. Transitions to existing HUD view (same page.tsx layout but now with live data)

### Screen 2: Camera + Live HUD (existing Phase 1 layout)
- Same layout as Phase 1 but TxLineProvider now fetches from real API
- Added: **API connection indicator** ‚Äî small pill top-right or inset in Scoreboard
- Added: **Error/offline overlay** ‚Äî translucent banner or full overlay when API unreachable

### Screen 3: Error / Offline (replaces camera or HUD)
- Triggered when: API unreachable on initial load, camera error, both offline
- Optional offline shell from service worker (if cached)

### State: API Connection

| State | Visual | Behavior |
|-------|--------|----------|
| `connecting` | Indicator dot: amber, pulsing glow-sm amber | Shown briefly on initial connect |
| `connected` | Indicator dot: cyan, steady glow-sm cyan | Normal operation |
| `reconnecting` | Indicator dot: amber, pulsing glow-md amber | Transient disconnect, auto-retry |
| `disconnected` | Indicator dot: magenta, steady glow-sm magenta | API unreachable after all retries |
| `stale` | Indicator dot: amber, slow pulse | Last packet >30s old, data may be outdated |

On `disconnected` ‚Üí show DataErrorBanner with "Retry" button. If camera was active, keep camera running, show data as last known (or dim).

On `stale` ‚Üí amber banner overlay "Stale Data" with last-updated timestamp, keep showing last packet.

### State: Match Selection

| State | Visual | Behavior |
|-------|--------|----------|
| Loading matches | Skeleton cards (3-4x), pulsing shimmer | `animate-pulse` on card placeholders |
| Matches loaded | Scrollable match list | Each card: teams, score, minute, live indicator |
| Empty (no matches) | Empty state card: "No Live Matches" + body + pull-to-refresh hint | Centered in viewport |
| Error loading | Error state card: "Connection Lost" + "Retry" button | Centered with amber border/accent |
| Selected (tap) | Card highlights with cyan border + glow-sm, starts camera transition | 200ms scale-in transition to camera |
